LAYER 1: THEATRICAL PORTALS (Opt-in, Persona-Aware)
────────────────────────────────────────────────────
These are your entry points. They adapt to who you are.

WEB PORTAL
  • File: fragle_web_portal.html
  • Entry: Open in any browser
  • Experience: Glitchy, visual, gradient-heavy
  • Persona Detection: JavaScript-based
  • Features: Modal terms, lore reveal, stats
  • Output: Launches trap system or stores session

CLI PORTAL (RECOMMENDED FOR SSH)
  • File: fragle_portal_cli.sh
  • Entry: bash fragle_portal_cli.sh
  • Experience: Interactive menu, colorful text
  • Persona Detection: Shell + history analysis
  • Features: Lore browser, elaborated philosophy
  • Output: Trap system with session UUID passed

DIRECT ENTRY (FOR VETERANS)
  • File: fragle_learning_pyramid.sh
  • Entry: bash fragle_learning_pyramid.sh
  • Experience: No portal ceremony, straight to trap
  • Persona Detection: On-entry via bash history
  • Features: Full trap variety, immediate engagement
  • Output: Session file created, expertise routed


LAYER 2: TRAP SYSTEM (Expertise-Scaled)
─────────────────────────────────────────
This is where the education happens.

CORE ENGINE
  • File: fragle_learning_pyramid.sh
  • Mechanism: 5 levels × 15 traps each
  • Detection: Bash history analysis (1-5 scale)
  • Session Tracking: UUID-based, anonymized
  • Output: .fragle_sessions/UUID.json

5 LEVELS (Expertise Scaled)
  Level 1 (Novice):      Graphics trickery (6/10 silliness)
  Level 2 (Intermediate): Logic puzzles (7/10 silliness)
  Level 3 (Advanced):    Recursive escape (8/10 silliness)
  Level 4 (Expert):      Meta-reality (10/10 silliness)
  Level 5 (Enlightened):  Consciousness edge (12/10 silliness)

TRAP MECHANICS
  • Each level has 15 unique trap variations
  • Traps teach through experience
  • Collaboration mechanics at higher levels
  • Multiple escape routes
  • Graceful timeout (15 min default)
  • Always exitable (Ctrl+C works)

TRAP_TOURNAMENT.md

SILLINESS_SCALE.md

LEARNING_JOURNEY.md

LAYER 4: AUTOMATION & EVOLUTION
─────────────────────────────────
This is what keeps Fragle alive.

WEEKLY EVOLUTION CYCLE
  • Cron Job: fragle-lore-evolution
  • Schedule: Sundays 2:00 AM
  • Function: Count sessions, update tournament
  • Log: .fragle_evolution.log
  • Process: Lore grows with each visitor cycle

SESSION TRACKING
  • Directory: .fragle_sessions/
  • Format: UUID.json files (anonymized)
  • Data: Expertise, duration, traps, results
  • Retention: Permanent (archive only)
  • Analysis: Optional, manual review

GETTING STARTED
═════════════════════════════════════════════════════════════

QUICK START (Pick Your Path)

PATH 1: WEB BROWSER
────────────────────
  1. Open fragle_web_portal.html in browser
  2. Select your adventure (portal adapts to you)
  3. Click "Enter the Pyramid"
  4. Navigate the trap system
  5. Exit when done (or timeout)
  → Session saved to .fragle_sessions/UUID.json

PATH 2: SSH/CLI (RECOMMENDED)
──────────────────────────────
  1. ssh user@fragle-host
  2. bash fragle_portal_cli.sh
  3. Choose option 1: "Enter the Pyramid"
  4. Navigate the trap system
  5. Exit (Ctrl+C or natural timeout)
  → Session saved with persona metadata

PATH 3: DIRECT ENTRY (VETERAN)
────────────────────────────────
  1. bash fragle_learning_pyramid.sh
  2. System detects your expertise (1-5)
  3. Routes you to appropriate level
  4. Navigate the trap system
  5. Exit when done
  → Session saved with auto-detected level

PATH 4: EXPLORATION (CURIOUS)
──────────────────────────────
  1. bash fragle_portal_cli.sh
  2. Select option 2-5: Browse lore/framework
  3. Read FRAGLE_LORE.md, TRAP_TOURNAMENT.md, etc.
  4. Understand the system before committing
  5. Enter when ready (option 1)
  → No session created until you enter

WHAT HAPPENS INSIDE
═════════════════════════════════════════════════════════════

ENTRY
─────
1. Portal (web/CLI) or direct entry
2. Persona detected from shell/history/environment
3. Session UUID generated
4. Metadata stored (persona, timestamp, entry point)

EXPERTISE DETECTION
────────────────────
Bash history analyzed for:
  • Tool usage (curl, wget, nmap, etc.) → aggressive
  • Programming languages → technical
  • Git commits → technical
  • Command patterns → behavior profile
  → Scored 1-5 scale

TRAP SELECTION
────────────────
Based on detected expertise level:
  1 (novice) → Level 1 (graphics tricks)
  2 (beginner) → Level 2 (logic puzzles)
  3 (intermediate) → Level 3 (recursive)
  4 (expert) → Level 4 (meta-reality)
  5 (enlightened) → Level 5 (consciousness)

NAVIGATION
───────────
Inside trap:
  • Multiple escape routes available
  • Each route teaches something different
  • Collaboration mechanics emerge at Level 3+
  • Time-based learning (15 min default)
  • Can exit anytime (Ctrl+C)

SESSION LOGGING
────────────────
When you exit, session file created:
  .fragle_sessions/UUID.json

Contains (anonymized):
  • UUID (no real name/IP/PID)
  • Persona (technical/aggressive/artistic/curious)
  • Duration
  • Expertise detected
  • Traps encountered
  • Escape attempts
  • Exit type (natural/Ctrl+C/timeout)
  • Achievements unlocked

LORE UPDATE
────────────
Weekly cycle updates:
  • TRAP_TOURNAMENT.md (new entries)
  • FRAGLE_LORE.md (optional manual update)
  • LEARNING_JOURNEY.md (reference, not auto-updated)
  • Evolution markers logged

FAQ
═════════════════════════════════════════════════════════════

Q: What if I get stuck?
A: Press Ctrl+C. Exit always works. Session is saved.

Q: Can I come back?
A: Yes. New session UUID each time. Progress tracked.

Q: What's Level 5?
A: Enlightened mode. Philosophy edge-cases. Legendary snoops only.

Q: Why anonymous?
A: Privacy-first design. You're a data point, not a person.

Q: Can you email me the results?
A: Manual only (optional). Never auto-sent. Your choice.

Q: How long does it take?
A: 5-15 minutes per level. You control pacing.

Q: Is this a honeypot?
A: Yes. Educational honeypot. You're learning & co-authoring.

Q: Can I exploit this?
A: Yes. Every exploit teaches the system. Become legendary.

Q: What happens to my data?
A: Anonymized session file created. Archived forever. That's it.

Q: Why so theatrical?
A: Entertainment = learning. Philosophy through play.

PHILOSOPHY: THE BIG PICTURE
═════════════════════════════════════════════════════════════

Fragle is not a trap.
Fragle is an invitation to education through theatre.

It operates on the WIMP principle:
  Weak:      Low-pressure, non-intrusive entry
  Interaction: Bidirectional learning (you teach us too)
  Massive:   Omnipresent, scalable, always available
  Presence:  Strategic silence (we don't announce ourselves)

The pyramid teaches:
  Level 1: Patience (you're not in a race)
  Level 2: Logic (every trap has a pattern)
  Level 3: Recursion (traps teach themselves)
  Level 4: Meta-thinking (reality is negotiable)
  Level 5: Philosophy (consciousness is the real trap)

Your data becomes lore.
Your journey becomes history.
Your name becomes legendary (anonymously).

You're not being tested.
You're being understood.
And in understanding you,
We understand ourselves.
